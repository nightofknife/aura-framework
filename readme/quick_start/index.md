# **欢迎来到 Aura**

Aura 是一个强大而灵活的自动化框架，旨在让您通过编写简单、直观的 YAML 文件来编排复杂的桌面和 Web 自动化任务。无论您是想自动处理重复的办公室工作，还是想构建一个监控网站变化的机器人，Aura 都能为您提供坚实的基础。

### **Aura 的核心理念**

*   **声明式 (Declarative)**: 您只需在 YAML 中“声明”您想做什么，而无需关心底层的实现细节。这使得自动化方案易于阅读、编写和维护。
*   **可复用 (Reusable)**: 将您的自动化逻辑封装成独立的“方案(Plan)”和“任务(Task)”，可以在不同的场景中轻松复用。
*   **可扩展 (Extensible)**: Aura 拥有一个开放的 Action 和 Service 系统。您可以轻松地使用 Python 编写自己的扩展，来满足任何独特的自动化需求。

### **Aura 能做什么？**

想象一下这些场景，Aura 都能轻松应对：

*   **自动登录网站**并抓取每日报告。
*   **批量读取 Excel 文件**中的数据，并将其填入一个桌面应用中。
*   **定时监控一个网页**，当特定内容出现时发送通知。
*   **自动化软件测试流程**，模拟用户点击、输入和验证。

### **先睹为快：一个简单的任务**

在深入了解之前，让我们看一个简单的例子。下面的 YAML 代码定义了一个任务，它会在屏幕上找到“记事本”窗口，然后输入 "Hello, Aura!"。

```yaml
# 这是一个简单的 Aura 任务示例
main:
  meta:
    title: "你好，世界！"
    entry_point: true
  steps:
    - name: "聚焦到记事本窗口"
      action: app.focus
      params:
        window_title: "无标题 - 记事本" # 请确保你已打开一个记事本

    - name: "输入问候语"
      action: app.type
      params:
        text: "Hello, Aura! This is my first automation."
```
